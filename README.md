## 环境要求

- JDK 17+
- Maven 3.6+
- Spring Boot 3.5+

## 快速开始

1. 克隆项目
2. 配置 `application.yml` 中的 API Key
3. 运行 `mvn spring-boot:run`

详细教程请查看各阶段目录下的 README.md 文件。
